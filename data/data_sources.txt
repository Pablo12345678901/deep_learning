Here are the sources of the data to be placed into the 'data' directory for the scripts to work.

Miscellaneous data from the author of the deep learning cookbook :
- from : https://github.com/DOsinga/deep_learning_cookbook
- download : copy/paste the below command in your terminal and replace the data directory path by yours :
git clone https://github.com/DOsinga/deep_learning_cookbook && cp -t THE-DATA-DIR-FROM-THIS-REPO-NOT-THE-ONE-FROM-THE-AUTHOR-REPO deep_learning_cookbook/data/*
